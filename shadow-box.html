Shadow Boxes

Nest your divs and label them outside in: box-one, box-two, box-three, and box-four. The CSS is as follows:

                        
.shadow-box .box-one {
    width: 400px;
    height: 350px;
    margin: 0 auto;
    padding-top: 50px;
}
.shadow-box .box-two {
    width: 300px;
    height: 250px;
    margin: 0 auto;
    padding-top: 50px;
}
.shadow-box .box-three {
    width: 200px;
    height: 150px;
    margin: 0 auto;
    padding-top: 50px;
}
.shadow-box .box-four {
    width: 100px;
    height: 100px;
    margin: 0 auto;
}
.shadow-box .shadow {
    -moz-box-shadow:    0px 0px 20px 20px #FAFAFA;
    -webkit-box-shadow: 0px 0px 20px 20px #FAFAFA;
    box-shadow:         0px 0px 20px 20px #FAFAFA;
}
